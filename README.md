# 🏦 Banking System (C++)

## 📌 Overview

A banking system using OOP concepts to manage accounts and transactions.

---

## ⚙️ Features

* Customer management
* Account creation
* Deposit / Withdraw
* Fund transfer
* Transaction history
* File storage

---

## ▶️ How to Run

### 🔹 Run using EXE

Double-click:
banking_system.exe

### 🔹 Run using Compiler

g++ banking_system.cpp -o bank
./bank

---

## 📁 Files

* banking_system.cpp
* banking_system.exe
* customers.dat
* accounts.dat
